Descomplicando a programação

    Cliente x Servidor

  • Cliente

    • Navegador (Browser)

  • Servidor

    • Computados em algum lugar do mundo que tem os códigos

  • Troca de Dados (arquivos)

    • Cliente faz a requisição e o servidor executa e retorna ao cliente

    Cliente é o front-end e o servidor o back-end

    Técnologias Front-end

  • HTML

    • Liguagem de marcação de texto para estruturar os textos, criar links, imagens etc...

  • CSS

    • Linguagem de estilo para deixar o HTML bonito

  • JavaScript

    • Liguagem de programação que funciona no navegador

    Tecnologias Back-end

  • NodeJS

    • Rodar o JavaScript no computador

  • SQL

    • Banco de dados para proteger os dados do seu programa
